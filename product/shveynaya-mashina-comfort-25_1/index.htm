Характеристики и описание товара

Швейная машина Comfort 25 – простое устройство с классической конструкцией. С ее помощью можно оперативно выполнить пошив изделий. Модель оснащается вертикальным челноком. В набор поставки входит педаль, с ее помощью происходит регулировка скорости. Comfort 25 оснащается металлической рамой. Присутствует в конструкции лапкодержатель и челночный узел. Модель можно полноценно применять с различными видами тканей. При использовании не возникает трудностей, изделие справится со всеми задачами. Лапка поднимается на 14 мм. Можно настроить длину и ширину стежка для полноценного пошива изделий. Выполняется одна петля для фиксации. Устройство осуществляет 12 швейных операций. Оно полуавтоматическое, существенно упрощается работа с оснащением. Есть плавная регулировка скорости шитья. Вы сможете подобрать подходящий параметр для последующей работы с оснащением. Производитель встроил элемент для освещения, удается легко контролировать процесс выполнения операций. При использовании модель потребляет 85 Вт, подключение происходит к сети. Установлена ручка для комфортного перемещения.
