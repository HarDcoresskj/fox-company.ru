Характеристики и описание товара

Кулер для процессора ID-Cooling SE-913-B PWM способен помочь гарантировать безопасные условия эксплуатации системного блока высокого класса. Устройство рассеивает мощность до 130 Вт. Эффективность кулера помогают обеспечивать 3 медные тепловые трубки. Кулер для процессора ID-Cooling SE-913-B PWM имеет башенную конструкцию. Не слишком значительная (122 мм) высота устройства не способствует появлению сложностей в процессе выбора корпуса. Кулер легкий: его масса равна 370 г. В оснащение кулера входит вентилятор с прозрачными лопастями. Форм-фактор вентилятора – 92 мм. Можно установить второй вентилятор. Кулер работает очень тихо. Уровень шума при пиковой частоте вращения (2200 оборотов в минуту) равен 23.8 дБ. Минимальная частота вращения – 600 оборотов в минуту. При такой скорости детектировать работу кулера только по шуму можно только в абсолютной тишине окружающего пространства. Особенность внешнего вида модели – подсветка синего цвета. В комплектацию кулера входят крепления для сокетов, скобы для фиксации вентиляторов и термопаста.

Производитель оставляет за собой право изменять характеристики товара, его внешний вид и комплектность без предварительного уведомления продавца. Предложение по продаже товара действительно в течение срока наличия этого товара на складе
