Характеристики и описание товара

Лазерный уровень Instrumax Element 2D SET подходит для проведения строительных и ремонтных работ на улице и в помещении. Он служит для точного горизонтального и вертикального выравнивания поверхностей, создавая два перпендикулярных лазерных луча. Дальность действия лучей у этой модели составляет 10 метров, а использование приемника лазерного луча позволяет увеличить ее до 50 метров. Прибор позволяет достичь точности измерений ± 0.2 мм/м. Построитель лазерных проекций Instrumax Element 2D SET укомплектован штативом с выдвижной колонкой, с помощью которого его можно разместить на высоте 35-63 см. Для фиксации на штативе или другом креплении прибор снабжен резьбой 1/4" на корпусе. Питаясь всего от двух батареек типа АА, уровень с черно-голубым пластиковым корпусом может непрерывно работать до 48 часов.
